## 👨‍🔬 About Me

I am a **Master's researcher** in the field of **Remote Sensing, SAR image processing, and Deep Learning**. My work focuses on applying modern deep learning architectures to solve real-world challenges in synthetic aperture radar (SAR) autofocus and terrain classification.

- 🛰️ **Research area:** SAR image formation, autofocus, and terrain classification  
- 🧠 **Expertise:** Deep Learning · CNN · U-Net · Res-U-Net · Image Processing  
- 📄 **Published:** [JARS 2024 — Enhanced SAR autofocus and classification using 2D SARNet](https://doi.org/10.1117/1.JRS.18.026507)  
- 🌍 **Location:** Egypt  
- 🎓 **Degree:** Master of Science (in progress)

## 🗞️ Featured Publication

> **Enhanced synthetic aperture radar image autofocus and classification using 2D SARNet framework**  
> *Mohamed Sakr, Ahmed Saleh, Fathy AbdElkader, Ghada Amer, Mohamed AboElenean*  
> 📰 **Journal of Applied Remote Sensing (JARS)**, 2024  
> 🔗 DOI: [10.1117/1.JRS.18.026507](https://doi.org/10.1117/1.JRS.18.026507)

**Key contributions of the paper:**
- ✅ ResU-Net based SAR image autofocus pipeline
- ✅ CNN terrain classifier (Mountain / Sand / River) — 100% test accuracy
- ✅ Near-real-time SAR processing workflow
- ✅ Evaluated with SSIM & PSNR metrics
